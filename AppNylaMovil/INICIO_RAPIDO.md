---

## 🚀 PASOS RÁPIDOS

### 1️⃣ Crear Iconos (5 min)

**Opción A - Generador Online (MÁS FÁCIL):**
```
1. Abre: https://www.pwabuilder.com/imageGenerator
2. Sube tu logo o cualquier imagen
3. Descarga el paquete de iconos
4. Copia a la carpeta public/:
   - icon-192x192.png
   - icon-512x512.png
```

**Opción B - Placeholder Rápido:**
```
1. Busca cualquier imagen cuadrada en Google
2. Descárgala
3. Redimensiona a 192x192 y 512x512
4. Guarda en public/ con esos nombres
```

### 2️⃣ Verificar (1 min)
```bash
npm run verify-pwa
```

Debe mostrar:
```
✅ Pasados: 8
❌ Fallidos: 0
```

### 3️⃣ Probar (2 min)
```bash
npm run dev
```

Abre: http://localhost:3000

### 4️⃣ Instalar en Móvil (2 min)
```bash
# 1. Obtén tu IP:
ipconfig

# 2. En tu teléfono, abre:
http://TU-IP:3000

# 3. Menú > "Agregar a pantalla de inicio"
```
